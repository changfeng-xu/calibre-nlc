# calibre-nlc
Calibre metadata download plugin based on [National Library of China](http://www.nlc.cn/).

## Calibre元数据下载插件
由于豆瓣的反爬虫限制，连续几次下载之后就会被限制访问，考虑增加[国家图书馆](http://www.nlc.cn/)信息查询接口。

## 插件安装
从release页面下载zip包，然后在calibre中安装为插件即可。
